# Academic Proofreader 

An NLP project trained on datasets derived from research papers, publications, and other academic materials. It analyzes input text to identify and correct spelling and grammatical errors, helping improve the accuracy and quality of academic writing.

>[!IMPORTANT]
>This project is a recreated version of the original system I developed as a school project. It was originally built in collaboration with team members, with shared contributions to development and deployment.
